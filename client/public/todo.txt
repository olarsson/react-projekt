Databas
------------------------------------

DB-namn:  users
id:	  [unique number, integer]
admin:	  [boolean]
email:	  [unique string, string ]
password: [password,      string ]


DB-namn: posts
id:	  [unique number, integer]
postedby: [*users.id, 	  integer]
text:	  [post content,  string ]
created:  [datetime,      integer]
likes:	  {
	    0: [*users.id],
	    1: [*users.id],
	    2: [*users.id]
	  }
